## Deploy to Vercel in 5 Minutes (Free)

### Step 1 — Create a GitHub account
Go to **github.com** and sign up (free).

### Step 2 — Create a new repository
1. Click the **+** icon → **New repository**
2. Name it `nestsync`
3. Set it to **Public**
4. Click **Create repository**

### Step 3 — Upload your files
1. Click **uploading an existing file**
2. Drag and drop both files:
   - `index.html`
   - `vercel.json`
3. Click **Commit changes**

### Step 4 — Deploy to Vercel
1. Go to **vercel.com** and sign up with your GitHub account (free)
2. Click **Add New Project**
3. Select your `nestsync` repository
4. Click **Deploy** — no settings to change
5. In ~30 seconds you get a live link like: `nestsync.vercel.app`
